# やりたいこと
* 暗記学習を手助けするサービス

# やりたいことの詳細、当初考えていること
* 内容を暗記した後でそれを可能な限り書き出す学習法がある
* この学習法では以下の点でやりづらさがある
  * 書き出した内容が間違っている部分を探す
  * 書き出した内容のうち足りない部分（暗記できていない部分）を探す
* これを補助するサービスを作りたい
* そもそも書き出しでなくてもいいと考えている。音声入力に対応したい
* ユースケースのイメージ
  * 電車の中で資料を読み、電車を降りて目的地に向かうまでの徒歩中にサービスへと覚えた内容を音声入力。到着してから足りない部分や理解し間違えている部分を確認する
  * 別途、いわゆる紙に書き出す方法のサポートも行いたい

# システムの想定
## 進め方の想定
* まずはフロントエンドだけつくりたい
* 前回の記録を保存するようなバックエンド機能は第2弾でつくる
  * csvダウンロードくらいはあってもいい
* aiサービスのapikeyを入れれば使える、というイメージ
